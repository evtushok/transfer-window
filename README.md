# Футбольное трансферное окно

Консольная игра на C++, реализованная в рамках курсового проекта по ООП.

Игрок выступает в роли спортивного директора ФК «Спартак Москва» и управляет клубом через трансферные операции: покупка, продажа и аренда футболистов.

## Архитектура

UML-диаграмма классов находится в папке [`docs/`](docs/diagram_uml.png).

```
Person  (абстрактный)
├── Agent
└── Player

Date
Club
LoanInfo
TransferWindow
TransferManager
Market      (статический — рыночная логика)
UI          (статический — весь ввод/вывод)
Game        (координатор)
```

**Реализованные концепции ООП:**
- Наследование и виртуальные функции (`Person → Agent`, `Person → Player`)
- Перегрузка операторов: `++`, `--`, `-`, `<<`, унарный `-`
- Конструктор копирования и move-конструктор (`Player`)
- Статические методы и поля (`Market`, `UI`, `Player::getTotalPlayers`)
- Обработка исключений (`std::invalid_argument`, `std::runtime_error`)
- Указатели с ручным управлением памятью (`Agent*` внутри `Player`)

## Игровой процесс

**Трансферное окно:** 1 июня — 31 августа 2026. Покупка, продажа и аренда доступны только в этот период.

**Ежедневный цикл (команда «Следующий день»):**
1. Клуб получает 0.5 млн € от спонсоров
2. Рыночная стоимость каждого игрока случайно изменяется на ±10%
3. Проверяются сроки активных аренд
4. С вероятностью 15% поступает предложение от клуба-конкурента

**Аренда** — 1, 3, 6 или 12 месяцев (10%, 30%, 50%, 70% от стоимости игрока).

**Продажа** — вероятность согласия покупателя падает при цене выше рыночной:
| Наценка       | Вероятность |
|---------------|-------------|
| ≤ 0%          | 85%         |
| до +10%       | 65%         |
| +10% — +20%   | 45%         |
| > +20%        | 25%         |

**Стратегия:** покупать молодых игроков (до 24 лет — «перспективные», растут быстрее), продавать с прибылью или принимать офферы конкурентов.

## Сборка и запуск

### Требования
- Компилятор C++17: `g++` (GCC 7+) или `clang++`
- `make` (опционально)

### С Makefile
```bash
make
./prog
```

### Вручную
```bash
g++ -std=c++17 -Wall -O2 -o prog main.cpp
./prog
```

### Очистка
```bash
make clean
```
